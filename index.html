ADDITION

JS cannot access output devices. The only exception is window.print()

var

  • redeclaring a variable inside a block will also redeclare the variable outside the block
  • Variables declared with let have Global Scope
  • var can be redeclared multiple times in the same scope. that's why the chances of error increases.

let

  • redeclaring a variable inside a block will not redeclare the variable outside the block
  • Variables declared with let have Block Scope
  • Variables declared with let must be Declared before use
  • Variables declared with let cannot be accessed outside the block scope
  • Variables declared with let cannot be Redeclared in the same scope

is var and let both are hoisted? what is the difference?

Yes, both are hoisted. The difference is var is hoisted and it assigns the value as "undefined".
In case of let, the variable is hoisted ( brought to the top of code) but it's value is not set as "undefined" (means not initialized as var)

const

  • const is Block scoped
  • const cannot be re-assigned in the same scope
  • const must be assigned with a value when declared. no line break in allowed.
  • it should be used for declaring :
    1. A new array
    2. A new object
    3. A new Function
    4. A new RegExp
  • it dost not define a constant value. It defines a const reference to a value
  • const value, array, object cannot be re-assigned. But we can change a const array and obj.
  • You can change the elements of a const array, but u can not re-assign the array

Operators and Operands

  • The numbers (in an arithmetic operation) are called operands.
  • The operation (to be performed between the two operands) is defined by an operator.

Assignment Operators

x += 5

means x = x + 5

% means remainder.

let x = 10;

x %= 5; //means x= x/5, meaning x= 10/5, which has no remainder. so the answer would be 0 in this case.

Difference between Array and Object

Array elements are seperated by comma, written within square bracket.

Objects properties are written as name:value pairs, seperated by commas, written within curly braces.
